# DO GOOD, STAY BAD / 3AM CLUB — Annotated Prototype

This is a fully coded front-end prototype for a **Spooky Scary Storytime / AshleeInc inspired ecosystem**.

It keeps the existing feeling of Ashlee’s site: black background, orange/copper typography, campfire/night-sky energy, story submissions, Patreon, podcast, and shop. Then it adds interactive systems around it: Story Vault, Story Constellation, 3D Atlas, Vault tools, community rooms, workbooks, planners, and shop expansion.

## Open the project

1. Unzip the folder.
2. Open it in VS Code.
3. Install/open with **Live Server**.
4. Start with `index.html`.

## Best files to study first

1. `index.html` — how the homepage sections connect.
2. `css/style.css` — how the visual design works.
3. `js/main.js` — how all interaction works.
4. `data/stories.json` — how stories feed the cards, constellation, and globe.
5. `docs/HOW-EVERYTHING-CONNECTS.md` — full explanation map.

## Prototype features

- Animated smoke layer
- Mobile menu
- Scroll fade-in animation
- Story cards rendered from JSON
- Story submission form
- “You are not alone” matching concept
- Story constellation visualization
- Shop products rendered from JSON
- Add-to-cart prototype
- 3D globe with hover-to-pause story pins
- Local browser saving through localStorage

## Important

This is not yet a backend app. It is a front-end prototype. Saved entries live in your own browser only.
