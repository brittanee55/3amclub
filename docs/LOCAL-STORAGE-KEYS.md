# LocalStorage Keys Used

These are the browser storage buckets used in the prototype.

- `submittedStories`: Saves stories from `submissions.html`.
- `journalEntries`: Saves journal/vault entries.
- `fanTheories`: Saves theories from community pages.
- `accessRequests`: Saves email access requests.
- `cart`: Saves product names added to cart prototype.

To clear prototype data in the browser: open DevTools → Application → Local Storage → clear this site.
