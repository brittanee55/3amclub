# Open this website in VS Code

1. Unzip this folder.
2. Open VS Code.
3. Go to **File → Open Folder**.
4. Choose this folder.
5. Open `index.html`.
6. Install **Live Server** if needed.
7. Right-click `index.html` → **Open with Live Server**.

Final homepage files:
- `index.html`
- `css/style.css`
- `assets/images/site/homepage-campfire-smoke-final.png`
- `assets/images/site/homepage-final-preview.png`
